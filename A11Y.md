# Accessibility

## Keyboard Navigation

- Arrow keys (left/right) navigate between slides in both edit and
  preview modes
- All interactive elements (buttons, inputs) are accessible via Tab key
- Modal dialogs can be closed with Escape key (built into shadcn Dialog)

## Semantic HTML

- Headings (h2) used for presentation title
- Buttons use native 'button' elements, not divs
- Images include alt text descriptions (user-provided via alt field)
- Form inputs have associated Label components

## Visual Indicators

- Disabled arrow buttons have reduced opacity (opacity-30) to indicate
  non-interactive state
- Selected elements have a visible blue border
- Current slide is highlighted with blue ring in slide list
- Interactive elements have hover states (hover:scale-110 on arrows)

## Color and Contrast

- UI uses high-contrast text (black on white)
- Error messages use red (destructive variant) for clear visibility
- Slide numbers visible in gray at bottom-left of each slide
